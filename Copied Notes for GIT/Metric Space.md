```ad-Definition
title:Definition: Metric Space
A metric space is a pair $(X,d)$ where $d:X\times X\to[0,\infty)$ such that:
- For all $x,y\in X$, $d(x,y) = d(y,x)$
- For all $x,y\in X$, $d(x,y) = 0$ iff $x=y$.
- (Triangle Inequality) For all $x,y,z\in X$: $$d(x,y) \leq d(x,z) + d(z,y)$$


If all but the 2nd condition holds, then $(X,d)$ is a <u>pseudo-metric</u>.
```

```ad-Definition
title:Definition: Distance to a Set
Let $A\subseteq X$ and $x\in X$, define: 
$$d(x,A) = \inf_{y\in A}\ d(x,y)$$
```
```ad-Definition
title:Definition: Open Ball
Let $x\in X$, define: 
$$B(x,r) = B_r(x)= \{y\in X: d(x,y) < r\}$$
```
```ad-Definition
title:Definition: Extended Metric Space
An <u>extended metric space</u> is a pair $(X,d)$ where $d:X\times X\to[0,\infty]$ such that $E_{<\infty} = \{(x,y): d(x,y)<\infty\}$ is an equivalence relation, and for each $x\in X$, $([x]_{E_{<\infty}},d\rest ([x]_{E_{<\infty}})^2)$ is a metric space. Inuitivley, this is a formalization of a metric space that can have points $\infty$ apart.

An  <u>extended pseudo metric space</u> is defined similarly, except that $([x]_{E_{<\infty}},d\rest ([x]_{E_{<\infty}})^2)$ is allowed to be a pseudo-metric space.
```

```ad-Definition
title:Definition: Metric Space Topology
The metric space structure $(X,d)$ induces a [[Topological Space|topological space]] $(X,\tau)$ on $X$ generated by open balls, ie a set $O\in \tau$ iff for all $x\in O$ there exists $\eps > 0$ such that $B(x,\eps)\subseteq O$.
```
